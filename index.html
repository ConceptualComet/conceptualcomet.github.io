Name

Like a comet, my interests burn bright for a brief time before they fade into inky blackness, only to return once more on an orbiting path. That point when a comet reaches its shortest distance to the sun is called the perihelion.

Design

I wanted to capture the idea of ephemera collected over eons by an entity observing the Earth and delighting in human culture. Of course, space-related items would feature heavily. I purposely rejected simple design and y2k nostalgia-bait, instead embracing my own style of eclectic maximalism to make a site that feels fully me.

Technology

This is the first website I've totally built myself since high school (class of 07!), so I kept it very simple on the technology side. I handcode directly on GitHub in the browser. Yep, even the blog entries. I use Claude Code to audit for accessibility. The site is hosted on GitHub Pages. The music player is Amplitude.js. My domain name registrar is Porkbun. I don't track analytics or use cookies, etc., and there's no comment section or guestbook. This is a personal space, not a platform, so I intentionally provide only an email address.
